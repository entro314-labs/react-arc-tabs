## Arc Tabs HTML Preview

This folder contains a standalone preview page to visually inspect the arc seam behavior (including the inverted bottom curves).

### What to look for

- Click the **middle tabs** to clearly see both left and right inverted curves.
- Use the **Radius** and **Gap** sliders to verify curve smoothness at different geometries.

### Run

1. Build the package so latest styles are available in `src/ArcTabs.css` / `dist/ArcTabs.css`.
2. Serve the repository root with any static server.
3. Open `/preview-html/src/` in your browser.
