# Security Policy

## Supported Versions

Security updates are provided for the latest published version of `@entro314labs/react-arc-tabs`.

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub Security Advisories:

- https://github.com/entro314-labs/react-arc-tabs/security/advisories/new

If you cannot use the advisory flow, open a private channel with the maintainers and include:

- a clear description of the issue
- impact and affected versions
- reproducible steps or proof-of-concept
- suggested mitigation (if available)

Please do **not** open public issues for security vulnerabilities.
